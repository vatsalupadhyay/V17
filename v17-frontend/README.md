# V17 Frontend

Angular application for V17 Clean Energy Dashboard.

## Setup

```bash
npm install
ng serve
```

App runs on http://localhost:4200

## Build

```bash
ng build
```

Output in `dist/v17-frontend/browser/`

```bash
ng e2e
```

Angular CLI does not come with an end-to-end testing framework by default. You can choose one that suits your needs.
